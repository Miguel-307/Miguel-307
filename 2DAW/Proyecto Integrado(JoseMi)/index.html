Preguntas frecuentes

¿Puedo usar solo un módulo (por ejemplo, Reservas)?

Sí. Activas el módulo que quieras y el resto queda deshabilitado. Puedes añadir más cuando lo necesites.

¿Necesito cambiar mi web actual?

No. Mijesux expone una API y widgets que puedes incrustar en tu web. También podemos crear tu web si lo prefieres.

¿Dónde se guardan mis datos?

En servidores europeos con copias de seguridad programadas. Exporta tus datos cuando quieras (CSV/JSON).
