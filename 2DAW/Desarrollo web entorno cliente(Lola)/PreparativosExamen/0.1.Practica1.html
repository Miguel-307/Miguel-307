🏋️‍♀️ Gimnasio de Sintaxis y Errores Tontos

No escribas todo. Solo detecta qué falta en los huecos (___).
Estos son los fallos que siempre nos bloquean.

1. Seleccionar un elemento por ID
El fallo clásico: ¿Es Element o Elements? ¿Es ById o ByID?
const titulo = document.get_______By__("miTitulo");
const titulo = document.getElementById("miTitulo");
[cite_start]

👉 Clave: Es 'Element' (singular) y 'Id' (I mayúscula, d minúscula). [cite: 304]

2. Escuchar un Clic
Olvidamos cómo se escribe el "escuchador".
boton.___Event________("click", () => { ... });
boton.addEventListener("click", () => { ... });

👉 Clave: 'add', luego 'Event', luego 'Listener'. [cite_start]Todo junto, camelCase. [cite: 145]

3. Recorrer lista para crear HTML
Queremos hacer algo por cada elemento. ¿Cuál es el método?
lista.for____( item => { ... });
lista.forEach( item => { ... });

👉 Clave: La E de 'Each' es mayúscula. [cite_start]No es map, map es para transformar, forEach es para recorrer. [cite: 654]

4. Convertir Objeto a Texto
Tienes un objeto y quieres enviarlo o guardarlo.
let texto = JSON._________(miObjeto);
let texto = JSON.stringify(miObjeto);

👉 Clave: stringify (hacer string). [cite_start]Si fuera al revés (texto a objeto) sería parse. [cite: 569]

5. Herencia de Clases
Quieres que Perro herede de Animal.
class Perro _______ Animal { ... }
class Perro extends Animal { ... }

👉 Clave: extends (extiende). [cite_start]Es la palabra mágica para la herencia. [cite: 406]

6. Evitar recarga al enviar
El formulario se envía y la página se pone blanca. ¡Falta una línea!
form.addEventListener("submit", (e) => {
  e.prevent_________;
});
e.preventDefault();

👉 Clave: Default (con D mayúscula) y ¡LOS PARÉNTESIS ()!. [cite_start]Es un método, si no pones (), no hace nada. [cite: 83]

7. Escribir texto dentro de un DIV
Tienes el elemento seleccionado en 'caja'. ¿Cómo le metes texto?
caja.text_______ = "Hola";
caja.textContent = "Hola";

👉 Clave: textContent (Contenido de texto). [cite_start]También vale innerText, pero cuidado con innerHTML (ese es para meter etiquetas). [cite: 298]

8. Ejecutar algo cada 1 segundo
Un reloj o cuenta atrás. ¿Cuál de los dos 'Set' era?
set________( () => console.log("Hola"), 1000);
setInterval( ... , 1000);

👉 Clave: Interval (Intervalo/Repetitivo). [cite_start]Timeout es para una sola vez (Tiempo fuera). [cite: 5801]

9. Quedarse solo con los aprobados
Tienes notas [5, 3, 8, 9]. Quieres un nuevo array solo con los > 5.
let aprobados = notas.______( n => n >= 5 );
let aprobados = notas.filter( n => n >= 5 );

👉 Clave: filter (Filtro). [cite_start]Devuelve un array nuevo más pequeño. [cite: 654]

10. Encontrar un objeto específico
Tienes una lista de alumnos. Quieres encontrar al que se llama "Pepe". (Solo el primero).
let pepe = alumnos.____( a => a.nombre === "Pepe");
let pepe = alumnos.find( a => a.nombre === "Pepe");

👉 Clave: find (Encontrar). Devuelve el objeto, no un array. [cite_start]Si usaras filter, te devolvería un array con un solo Pepe dentro. [cite: 654]
