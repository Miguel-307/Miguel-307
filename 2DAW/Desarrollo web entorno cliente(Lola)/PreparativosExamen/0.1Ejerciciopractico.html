🧠 Gimnasio de Lógica

Reto 1: El FizzBuzz

Introduce un número. La lógica debe decir:

  • "Fizz" si es divisible por 3.
  • "Buzz" si es divisible por 5.
  • "FizzBuzz" si es divisible por ambos (ej: 15).
  • El mismo número si no es divisible por ninguno.

Reto 2: El Espejo (Palíndromos)

Escribe una palabra. El sistema debe decirte si se lee igual al revés (ej: "ana", "reconocer").

Pista: Intenta invertir el string y compararlo con el original.

Reto 3: Encontrar el Mayor

Introduce números separados por comas (ej: 10, 5, 80, 2). Tu lógica debe encontrar el más grande sin usar Math.max().

Pista: Asume que el primero es el mayor, recorre los demás y si encuentras uno más grande, actualiza tu variable 'mayor'.
