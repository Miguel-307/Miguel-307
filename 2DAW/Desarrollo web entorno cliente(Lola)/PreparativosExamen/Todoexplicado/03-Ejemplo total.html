Tema 2: Control de Flujo

Condicionales

Permiten tomar decisiones en el código.

  • if (condicion) { ... } else { ... }
  • switch (valor) { case x: ... break; }
  • Ternario: condicion ? valorSi : valorNo

Demo: Switch

Bucles

Repiten bloques de código.

  • for (let i=0; i<n; i++): Bucle contador clásico.
  • while (condicion): Repite mientras sea true.
  • do...while: Ejecuta al menos una vez.

Tema 3: Estructuras de Datos

1. Arrays (Listas)

Colecciones ordenadas de elementos.

let frutas = ["Manzana", "Pera"]; frutas.push("Uva"); // Añadir al final let filtro = frutas.filter(f => f.length > 4); // Filtrar

2. Sets (Conjuntos)

Colecciones de valores únicos. No admite duplicados.

let unicos = new Set([1, 2, 2, 3]); // {1, 2, 3}

3. Maps (Diccionarios)

Pares Clave-Valor. La clave puede ser cualquier tipo.

let mapa = new Map(); mapa.set("admin", true); mapa.get("admin"); // true

Tema 4: Funciones

Tipos de Declaración

  • Declarativa: function suma(a,b) { return a+b; } (Tiene Hoisting).
  • Expresiva: const suma = function(a,b) { ... }.
  • Flecha (Arrow): const suma = (a,b) => a+b;.

Características Avanzadas

  • Parámetros Rest: function(...nums) agrupa infinitos argumentos.
  • Recursividad: Función que se llama a sí misma (ej: factorial).
  • Clausuras (Closures): Funciones que recuerdan su entorno léxico (variables privadas).

Tema 5: POO y Objetos Predefinidos

Clases y Herencia

Plantillas para crear objetos. Usan constructor, extends y super.

class Animal { constructor(n) { this.nombre = n; } } class Perro extends Animal { ladrar() { return this.nombre + " dice guau"; } }

JSON

Intercambio de datos.

  • JSON.stringify(obj): Objeto a Texto.
  • JSON.parse(txt): Texto a Objeto.

Objetos Nativos

  • Math: random(), floor(), max().
  • Date: new Date(), getFullYear().
  • RegExp: Patrones de búsqueda (ej: email). Método test().

Tema 6: BOM y DOM

BOM (Browser Object Model)

Interactúa con el navegador.

  • window.innerWidth: Ancho ventana.
  • navigator.userAgent: Datos del navegador.
  • location.href: URL actual.
  • setInterval(func, ms): Temporizador repetitivo.

DOM (Document Object Model)

Manipula el HTML y CSS.

  • Selección: getElementById, querySelector, querySelectorAll.
  • Creación: createElement, appendChild.
  • Estilos: elem.style.color, elem.classList.add().
  • Atributos: getAttribute, dataset (data-*).
  • Cookies: document.cookie.

Tema 7: Eventos

Interacción con el usuario. Se usa addEventListener('evento', funcion).

Tipos Comunes

  • Ratón: click, mouseover, mouseout.
  • Teclado: keydown, keyup (propiedad e.key).
  • Formulario: submit, change, focus, blur.

Propagación y Control

  • e.preventDefault(): Evita la acción por defecto (ej: enviar form).
  • e.stopPropagation(): Evita que el evento suba al padre (burbujeo).
  • e.target: El elemento que originó el evento.
